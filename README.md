# Gaadiwala
GaadiWala is a web-based Vehicle Rental Management System built using JSP, Servlets, Java, MySQL, HTML, CSS, JavaScript, and Bootstrap. It allows users to register, browse vehicles, and book rentals online, while admins can manage vehicles and bookings through a secure, responsive, and user-friendly platform.

Technology Stack
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: JSP, Servlets, Java
Database: MySQL
Server: Apache Tomcat 9
JDK: 17.0.9
